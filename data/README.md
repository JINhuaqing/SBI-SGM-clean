This folder contains demo data for tutorials, 

`example_meg.pkl` includes 

- sc_mat: sc matrix
- dist_mat: distance matrix
- ts: the original meg time series
- fc: the fc matrix from ts
- psd: the PSD from ts
- freqs: The freqs pts corresponding to psd
- fs: sampling freq (in Hz)


`example_fMRI.pkl` includes

- sc_mat: sc matrix
- ts: the original meg time series
